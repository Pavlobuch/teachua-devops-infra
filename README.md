# DevOps Portfolio Project

## Overview

This project demonstrates a complete DevOps workflow for deploying and operating a full-stack application in AWS.

The project includes Infrastructure as Code, Configuration Management, CI/CD, Containerization, Kubernetes orchestration, and Monitoring.

The main goal is to build a production-like environment using modern DevOps practices while keeping infrastructure costs low.

## Project Goals

* Deploy a full-stack application using Docker
* Automate infrastructure provisioning using Terraform
* Configure servers using Ansible
* Build CI/CD pipelines using Jenkins
* Deploy workloads to Kubernetes
* Implement monitoring with Prometheus and Grafana
* Follow DevOps best practices
